Abstract

Installation

CDN Development (redirect to latest):

<link rel="stylesheet" href="https://unpkg.com/hyper-slider-css@latest/dist/slider.min.css"/>

CDN Production (specific release):

<link rel="stylesheet" href="https://unpkg.com/hyper-slider-css@1.0.0/dist/slider.min.css"/>

NPM package:

$ npm install --save-dev hyper-slider-css

CSS import:

@import url("https://unpkg.com/hyper-slider-css@1.0.0/dist/slider.min.css");
